# 🚩 Hanuman Turtle Art

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/HANUMAN-TURTLE-ART.git
```

Go inside project

```bash
cd HANUMAN-TURTLE-ART
```

Install dependencies

```bash
pip install opencv-python numpy
```

Run the project

```bash
python turtle_draw.py
```

---

## ⚙️ How It Works

1. Read Hanuman Image
2. Convert image into edges using OpenCV
3. Detect contours
4. Python Turtle traces every contour
5. Complete Hanuman artwork is drawn automatically

The Turtle Graphics window gradually draws the complete artwork of Lord Hanuman.
